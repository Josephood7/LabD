## Build AXI UART Lite Test Hex
```sh
cd counter_la_uart
source run_xsim
```

## Open AXI UART Lite Vivado Block Design
```sh
cd vivado/vvd_caravel_fpga
vivado vvd_caravel_fpga.xpr
```
Click Open Block Design in left
